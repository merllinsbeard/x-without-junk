# Changelog

All notable changes to the News Parser Agent will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-29

### Added

- Initial release of News Parser Agent
- CLI commands: `timeline`, `bookmarks`, `user`, `search`, `analyze`, `status`, `interactive`
- Bird CLI integration for X/Twitter data fetching
- Content filtering system with configurable quality thresholds
- Markdown report generation for parsed tweets
- Claude AI integration for intelligent tweet analysis
- Environment-based configuration with `.env` support
- Rich terminal output with colored console and progress indicators
- MIT license

### Features

- Fetch home timeline and bookmarks
- Filter low-quality content (marketing, self-improvement)
- Parse and generate structured Markdown reports
- AI-powered analysis of tweet content
- Interactive CLI mode for batch operations
- System status and configuration verification
